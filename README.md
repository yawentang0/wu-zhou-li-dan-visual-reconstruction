# wu-zhou-li-dan-visual-reconstruction

中文名：《武周·皇嗣李旦》历史考据向影视化重构项目

这是一个以武周时期“皇嗣李旦”为核心对象的历史考据、视觉设定与影视化重构文档项目。项目目标不是先写剧本，而是先建立可靠的研究底座：人物、制度、礼仪、服饰、建筑、城市空间与政治语境都需要被清楚记录，再逐步转化为可用于影像创作的故事结构、视觉规则与制作管线。

## 项目定位

- 历史考据向：以史料、学术研究、图像材料与考古资料为基础，区分史实、推测与创作处理。
- 影视化重构：面向剧集、短片、概念影像或 AI 影像实验，沉淀故事圣经、分集大纲、场景表、镜头表和视觉提示词。
- 文档型 GitHub 项目：本仓库只保存 Markdown 文档、研究记录、设定说明与素材目录，不包含程序代码。

## 核心问题

- 李旦在武周政治结构中的真实处境是什么？
- “皇嗣”身份如何影响他的宫廷生活、政治选择与人物弧光？
- 武则天、李显、太平公主、李隆基、安金藏等人物与李旦之间的关系，如何在史实边界内被戏剧化呈现？
- 神都洛阳、明堂、宫城、朝会、礼仪与服饰如何形成可信的视觉世界？

## 目录说明

- `00_project_management/`：任务板、版本记录、决策记录。
- `01_research/`：史料、年表、人物索引、政治制度、礼仪服饰、建筑与城市研究。
- `02_characters/`：主要人物小传、关系与戏剧功能。
- `03_worldbuilding/`：明堂、宫城、街市、朝仪与视觉规则。
- `04_story/`：故事圣经、季纲、单集大纲、场景表与剧本草稿。
- `05_visual_development/`：角色、服饰、建筑、场景、分镜与图像提示词目录。
- `06_video_pipeline/`：动画、镜头、声音、音乐与工具测试记录。
- `assets/`：参考资料、生成图像、图表与导出文件。
